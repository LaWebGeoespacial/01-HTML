HTML

    El HTML (Hypertext Markup Language) es el lenguaje de marcas estándar para la creación de páginas y aplicaciones en la Web.

    Junto con CSS (Cascading Style Sheets) y Javascript, conforma el grupo de las tres tecnologías principales de la Web.

    Fue creado en 1990 por el físico inglés Tim Berners-Lee. La primera página web está aún disponible en http://info.cern.ch/

HTML

    El HTML especifica la estructura y la semántica de una página web.

    La presentación es especificada mediante CSS y la interactividad mediante JavaScript.

    Tanto HTML como CSS son estándares del World Wide Web Consortium (W3C).

HTML5

    Su especificación fue publicada por el W3C en 2014.

    Reduce la necesidad de complementos externos (ej. Flash) para el manejo de multimedia y otros recursos. Para esto, se apoya en CSS y JavaScript.

    Se diseñó para funcionar en diferentes tipos de dipositivos (ej. computadoras, tabletas, teléfonos).
